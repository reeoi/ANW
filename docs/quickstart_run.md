# ANP 正式使用指南（全 UI 版）

> 更新日期: 2026-05-08
> 适用版本: c_pipeline c1 + Phase 6 章节切分 + 执行控制台
> 默认目标平台: 番茄短故事 (单篇 8000-12000 字)

> **重要变化**：本指南面向「图形化全自动写作发布站」流程。
> 所有日常操作通过浏览器执行控制台完成，无需在 UI 中输入主题/风格/字数。
> 题材、风格、字数由 AI 从 `data/theme_pool.json` 自动选取。

---

## 0. 一次性准备（只做一次）

### 0.1 环境检查

```bash
cd D:/Development_alma/anp
.venv/Scripts/python.exe check_env.py
.venv/Scripts/python.exe check_config.py
```

两个脚本都应输出 OK / 无报错。如缺依赖:

```bash
.venv/Scripts/pip.exe install -r requirements.txt
.venv/Scripts/playwright.exe install chromium
```

### 0.2 关键密钥

`.env`（项目根）必须有:

```
DEEPSEEK_API_KEY=sk-...
```

UI 启动后也可在「设置 → 生成」里粘贴覆盖。

### 0.3 番茄登录态（首次必做）

启动 UI 后，进入「设置 → 发布」点击 **🔑 抓取登录态**：浏览器会弹窗，
完成番茄账号登录 → 关闭 → 状态卡变成 ✅ 有效。

## 2. 日常使用：执行控制台

打开「生成作品」标签页，顶部就是 **🎛 执行控制台**：

```
┌─ 执行控制台 ──────────────────────────────────────┐
│ [🚀 立即执行一次]  [⏹ 取消执行]  [⏰ 自动调度] │
│ 当前任务: phase_3_section_05 (story_id=42)         │
│ 下次时刻: 14:23 (剩 2h 12m) | 今日: 1/3 篇         │
│ 番茄登录: ✅ 有效 | 题材池: 87/100 | 调度器: ON    │
└────────────────────────────────────────────────────┘
```

### 2.1 「🚀 立即执行一次」

点一下，立刻启动一次「**生成 → AI 审核 → 番茄发布**」原子任务：

- AI 自动从题材池选题，跑完 Phase 0-6（约 7-8 分钟），
- 通过 AI 审核（不通过自动重写 Phase 4-5，最多 3 次），
- 通过后立刻发布到番茄。

按钮在任务期间变灰，下方「当前任务」实时显示当前 phase。

### 2.2 「⏰ 自动调度」

打开开关 → 后台调度器启动：

- 每天 03:00 自动从题材池中按「每日 N 篇 + 操作时段」抽样若干 slot；
- 到点先查库存 `status='approved'` 的作品；有则发布、没有则跑完整原子流程；
- 失败：3 次连续发布失败 → 该作品标 `needs_human` + 调度器自动暂停 + 顶部红色横幅。

关闭开关 → 取消未触发的 slot；已生成/已发布的不受影响。

### 2.3 「⏹ 取消执行」

只对正在运行的任务生效：

- 当前 phase 跑完后停止（不强制 kill LLM 调用）；
- 数据库 `status` 置为 `cancelled`，`current_phase` 置为 `cancelled_at_phase_N`；
- 已发布到番茄的作品**不能取消**（按钮变灰）。

### 2.4 监控信息（5 秒自动刷新）

下方 6 项实时刷新：

| 字段 | 含义 |
|---|---|
| 当前任务 | 正在跑的 story_id + phase |
| 下次时刻 | 最早的未触发 slot + 倒计时 |
| 今日发布 | 已发/计划数 |
| 番茄登录 | 登录态是否有效 |
| 题材池 | `data/theme_pool.json` 当前条数 |
| 调度器 | 是否运行 |

## 3. 审核 / 发布

「审核队列」tab 列出所有 pending / needs_human 作品。

「发布管理」tab 用于手动触发发布或验证 dry-run。

---

## 4. 故障排查

| 现象 | 处理 |
|---|---|
| 顶部红色横幅「连续 3 次发布失败」 | 「设置 → 发布」重新抓取登录态，刷新页面，切回「自动调度」即可恢复 |
| 「番茄登录」显示 ⚠ 失效 | 同上 |
| 「题材池」为 0 | 「生成作品」面板下方点「🌱 立即刷新题材池」 |
| 「立即执行」按钮一直灰 | 已有任务在跑；查看「当前任务」phase |
| 想从 CLI 测试 | `cli.generate` / `cli.batch_generate` 仍保留作为 power user 后门 |

---

## 5. 配置参考

主要文件：

- `config.yaml` — 调度时间、操作时段、每日 N 篇范围、AI 审核阈值等
- `.env` — DeepSeek / 番茄账号密钥
- `data/theme_pool.json` — AI 自动从此池选题，每周一 03:00 自动刷新
- `data/anp.sqlite3` — 作品状态、cost log、daily_publish_plan

不要在 UI 上找「默认主题/风格/字数」字段——已删除，由 AI 自动决策。
